Project Overview: The DeltaLumin Narrative site outlines the approach, implementation, and evaluation of a three month pilot product and service aimed at illuminating energy use for low-income Chicago residents.

The site was designed and developed as a part of General Assembly's Front-End Web Development course. The site includes interactive animated scalable vector graphics (svg) using CSS animations and data visualization using d3.js. It was built using the static site generator, Jekyll.

Visit the site at deltalumin.com/year-one-report.
